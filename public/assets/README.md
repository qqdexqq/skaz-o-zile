# Папка ассетов

Все изображения сайта берутся отсюда. Замените файлы-заглушки на свои,
сохранив имена и пути (или поправьте пути в `src/data/content.js`).

| Путь                     | Где используется                | Рекомендация             |
|--------------------------|---------------------------------|--------------------------|
| `portrait.png`           | Портрет героини (Hero)          | вертикальное фото 4:5    |
| `qr.png`                 | QR-код пожертвования (СБП)       | квадрат, ≥ 400×400 px    |
| `bg.jpg`                 | (опц.) свой фон-пергамент        | см. ниже                 |
| `venue/1.jpg … 5.jpg`    | Галерея локации (карусель)      | горизонтальные 4:3       |
| `looks/1.png … 4.png`    | Сетка образов дресс-кода        | вертикальные 3:4         |

Доступ по пути `/assets/...` (например, `/assets/venue/1.jpg`).

## Свой фон вместо процедурного пергамента
1. Положите файл `bg.jpg` в эту папку.
2. В `src/index.css` найдите блок `.parchment` и переключитесь
   на закомментированный вариант с `url('/assets/bg.jpg')`.
Фон адаптивный: `background-size: cover`.

## Сколько фото в галерее/сетке
Количество задаётся массивами `VENUE.gallery` и `DRESS_CODE.looks`
в `src/data/content.js` — добавляйте/убирайте элементы, сетка подстроится.
